# AG Calculator — WordPress Plugin

## Project Overview

**AG Calculator** — это универсальный WordPress-плагин для гибкого расчёта стоимости продукции с настраиваемой админ-панелью. Плагин позволяет создавать различные типы продуктов с динамическими полями ввода и формулами расчёта цены.

### Основные возможности

- **Custom Post Type** `ag_product` для управления продуктами
- **Динамические поля**: число, слайдер, выпадающий список, радио-кнопки, чекбокс, текст, скрытое число
- **Формульный движок** с поддержкой переменных, операторов и функций lookup
- **REST API** для расчёта стоимости на лету
- **Шорткоды** для встраивания калькулятора в страницы
- **Таблицы цен** для полей select/radio

### Технологии

- **Язык**: PHP 7.4+
- **CMS**: WordPress
- **Frontend**: Bootstrap 4, jQuery
- **Стили**: Custom CSS (admin + frontend)
- **JavaScript**: Vanilla JS + jQuery

## Building and Running

### Установка плагина

1. Скопируйте папку `ag-calc` в директорию:
   ```
   wp-content/plugins/ag-calc/
   ```

2. Активируйте плагин через админ-панель WordPress: **Плагины → AG Calculator**

3. Настройте продукты через меню **Калькулятор → Типы продукции**

### Шорткоды

| Шорткод | Описание |
|---------|----------|
| `[ag_calculator_tabs]` | Все продукты во вкладках |
| `[ag_calculator_product slug="product-slug"]` | Один продукт по слагy |

### REST API Endpoints

| Endpoint | Method | Описание |
|----------|--------|----------|
| `/wp-json/ag-calc/v1/products` | GET | Список всех продуктов |
| `/wp-json/ag-calc/v1/products/{slug}` | GET | Конфигурация продукта |
| `/wp-json/ag-calc/v1/calculate` | POST | Расчёт стоимости |

**Пример запроса расчёта:**
```json
POST /wp-json/ag-calc/v1/calculate
{
  "product_slug": "product-slug",
  "fields": {
    "pages": 20,
    "cover_type": "hardcover",
    "quantity": 2
  }
}
```

## Development Conventions

### Код-стайл

- **Именование классов**: Префикс `AG_` (например, `AG_Product_Manager`)
- **Именование методов**: snake_case для хуков, camelCase для логики
- **Переводы**: Текстовый домен `ag-calc`, функции `__()`, `_e()`, `sprintf()`
- **Безопасность**: Все данные санитизируются перед сохранением и экранируются при выводе

### Архитектурные паттерны

- **Singleton**: `AG_Field_Registry`, `AG_Formula_Engine`
- **Статические методы**: Для утилитных классов без состояния
- **Инстансы**: Для классов с зависимостями от хуков WordPress

### Типы полей

| Тип | Описание | Влияет на цену |
|-----|----------|----------------|
| `number` | Числовое поле с min/max/step | Да |
| `range` | Слайдер с min/max/step | Да |
| `select` | Выпадающий список с таблицей цен | Да |
| `radio` | Радио-кнопки с таблицей цен | Да |
| `checkbox` | Чекбокс с ценой при отметке | Да |
| `text` | Текстовое поле | Нет |
| `hidden_number` | Скрытое числовое значение из админки | Да |

### Формулы

Формулы используют синтаксис с переменными в двойных фигурных скобках:

```
(({{base_price}} * {{pages}}) + {{cover_price}}) * {{quantity}}
```

#### Пользовательские переменные

Можно определять свои переменные для сложных расчётов. Каждая переменная указывается с новой строки, основная формула — последней:

```
base_price = if({{kolvo}} >= 100, 120, 150);
discount = if({{sum}} > 5000, 0.1, 0);
{{kolvo}} * ({{base_price}} + {{razvorot}}) * (1 - {{discount}})
```

**Правила:**
- Имя переменной: латиница, цифры, подчёркивание (начинается с буквы)
- После имени `=` и выражение, заканчивающееся на `;`
- Переменные вычисляются сверху вниз, можно использовать ранее определённые
- В выражениях переменных можно использовать `{{field_key}}` и `if()`

**Примеры:**

```
// Оптовая цена при заказе от 100 шт
base_price = if({{kolvo}} >= 100, 120, 150);
{{kolvo}} * {{base_price}}

// Скидка от суммы заказа
sum = {{kolvo}} * {{price}};
discount = if({{sum}} > 10000, 0.15, if({{sum}} > 5000, 0.1, 0));
{{sum}} * (1 - {{discount}})

// Сезонный коэффициент (май-август)
season_coef = if({{month}} >= 5 && {{month}} <= 8, 1.2, 1);
{{base}} * {{season_coef}}

// Скидка для постоянных клиентов или оптовиков
discount = if({{is_vip}} == 1 || {{kolvo}} >= 50, 0.2, 0);
{{sum}} * (1 - {{discount}})
```

**Арифметические операторы:**
| Оператор | Описание | Пример |
|----------|----------|--------|
| `+` | Сложение | `{{price}} + 100` |
| `-` | Вычитание | `{{price}} - 50` |
| `*` | Умножение | `{{quantity}} * {{price}}` |
| `/` | Деление | `{{total}} / 2` |
| `()` | Скобки | `({{a}} + {{b}}) * {{c}}` |

**Математические функции:**
| Функция | Описание | Пример |
|---------|----------|--------|
| `min(a, b)` | Минимум из значений | `min({{price}}, 1000)` |
| `max(a, b)` | Максимум из значений | `max({{discount}}, 10)` |
| `pow(base, exp)` | Возведение в степень | `pow({{size}}, 2)` |
| `round(x)` | Округление | `round({{price}})` |
| `ceil(x)` | Округление вверх | `ceil({{area}})` |
| `floor(x)` | Округление вниз | `floor({{discount}})` |
| `abs(x)` | Модуль числа | `abs({{diff}})` |

**Логические операторы (для `if()`):**
| Оператор | Описание | Пример |
|----------|----------|--------|
| `&&` | И (оба условия истинны) | `{{qty}} > 10 && {{price}} < 100` |
| `\|\|` | ИЛИ (хотя бы одно истинно) | `{{vip}} == 1 \|\| {{qty}} >= 50` |
| `!` | НЕ (инверсия) | `!{{discount}}` |

**Пример сложной формулы:**
```
round(({{base_price}} * {{pages}} + {{cover_type}}) * {{quantity}})
```

### Переменные в формулах

| Тип поля | Переменная | Значение | Дополнительная переменная |
|----------|------------|----------|---------------------------|
| `number`, `range` | `{{field_key}}` | Числовое значение | — |
| `hidden_number` | `{{field_key}}` | Значение из конфига админки | — |
| `select`, `radio` | `{{field_key}}` | **Цена** выбранной опции | `{{field_key_value}}` — значение опции |
| `checkbox` | `{{field_key}}` | Цена если отмечен (0 если нет) | `{{field_key_value}}` — 1 или 0 |

**Пример формулы:**
```
{{pages}} * {{base_price}} + {{cover_type}} + {{gift_box}}
```
Где:
- `pages` — числовое поле (количество страниц)
- `base_price` — цена за страницу из select/radio
- `cover_type` — цена типа обложки из select/radio
- `gift_box` — цена опции checkbox (0 или цена)

### Безопасность

- Проверка nonce для всех форм
- Санитизация входных данных: `sanitize_text_field()`, `sanitize_textarea_field()`
- Экранирование вывода: `esc_html()`, `esc_attr()`, `esc_textarea()`
- Валидация формул перед вычислением
- Запрет опасных функций в `eval()`
