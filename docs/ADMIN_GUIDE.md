## 1. Установка и активация

### Требования
- WordPress 5.0+
- PHP 7.4+
- Тема с поддержкой Bootstrap 4 (или плагин подключит CDN автоматически)

### Установка
1. Скопируйте папку `foto-kniga-calculator` в:
   ```
   wp-content/plugins/foto-kniga-calculator/
   ```
2. Перейдите в **Плагины → Установленные**
3. Найдите **Foto Kniga Calculator** и нажмите **Активировать**

После активации в админке появится меню **Калькулятор**.

## 2. Создание продукта

Продукт — это единица продукции с собственными полями ввода и формулой расчёта цены.

1. Перейдите в **Калькулятор → Типы продукции**
2. Нажмите **Добавить новый**
3. Введите название продукта (например, «Фотокнига Стандарт»)
4. Заполните поля, описанные ниже
5. Нажмите **Опубликовать**

---

## 3. Настройки продукта

### Slug (ярлык)
Уникальный идентификатор продукта латиницей. Используется в шорткодах и REST API.

| Значение | Описание |
|----------|----------|
| `butterfly-photo` | Фотокнига «Бабочка» |
| `standard-album` | Стандартный альбом |
| `premium-box` | Премиум коробка |

> **Важно:** Slug должен быть уникальным. Используйте только латинские буквы, цифры и дефис.

### Иконка вкладки
Иконка из набора Dashicons для отображения во вкладках калькулятора.

**Популярные иконки:**
- `dashicons-calculator` — калькулятор
- `dashicons-images-alt` — изображения
- `dashicons-book` — книга
- `dashicons-star-filled` — звезда

### Заголовок вкладки
Текст, который отображается на вкладке калькулятора.

### Порядок сортировки
Число, определяющее порядок вкладок (меньше = раньше).

## 4. Поля калькулятора

Поля — это элементы ввода, которые видит пользователь при расчёте стоимости.

### Добавление поля
1. В блоке **Поля калькулятора** выберите тип поля из выпадающего списка
2. Нажмите **Добавить поле**
3. Заполните параметры поля
4. Перетаскивайте поля за иконку ☰ для изменения порядка

### Типы полей

#### Число
Поле ввода числового значения.

| Параметр | Описание |
|----------|----------|
| Ключ поля | Уникальный идентификатор (латиница, например `pages`) |
| Подпись | Текст, видимый пользователю |
| Минимум | Минимальное допустимое значение |
| Максимум | Максимальное допустимое значение |
| Шаг | Шаг изменения значения |

**Пример:** Поле «Количество страниц» с ключом `pages`, min=10, max=100, step=1.

#### Слайдер
Ползунок для выбора числового значения.

| Параметр | Описание |
|----------|----------|
| Ключ поля | Уникальный идентификатор |
| Подпись | Текст над слайдером |
| Минимум | Минимальное значение |
| Максимум | Максимальное значение |
| Шаг | Шаг перемещения ползунка |

**Пример:** Слайдер «Количество разворотов» с ключом `spreads`, min=5, max=50, step=1.

#### Выпадающий список
Пользователь выбирает один вариант из списка.

| Параметр | Описание |
|----------|----------|
| Ключ поля | Уникальный идентификатор |
| Подпись | Текст метки поля |
| Значения и цены | Список вариантов с ценаой для каждого |

**Пример:** Тип обложки с ключом `cover_type`:
| Значение | Цена |
|----------|------|
| Мягкая | 0 |
| Твёрдая | 500 |
| Премиум | 1500 |

> **Важно:** В формуле переменная `{{cover_type}}` будет содержать **цену** выбранного варианта, а `{{cover_type_value}}` — **текстовое значение**.

#### Радио-кнопки
Набор переключателей для выбора одного варианта.

Настройки аналогичны выпадающему списку.

**Пример:** Формат печати с ключом `format`:
| Значение | Цена |
|----------|------|
| 20×20 | 0 |
| 20×30 | 300 |
| 30×30 | 600 |

#### Чекбокс
Флажок для включения/выключения опции.

| Параметр | Описание |
|----------|----------|
| Ключ поля | Уникальный идентификатор |
| Подпись | Текст рядом с чекбоксом |
| Цена если отмечен | Дополнительная стоимость при активации |

**Пример:** «Подарочная упаковка» с ключом `gift_box`, цена = 300.

> **Важно:** В формуле `{{gift_box}}` = цена (0 или 300), `{{gift_box_value}}` = 0 или 1.

#### Текст
Текстовое поле для ввода произвольного текста. Не влияет на цену.

| Параметр | Описание |
|----------|----------|
| Ключ поля | Уникальный идентификатор |
| Подпись | Текст метки |

**Пример:** Поле «Комментарий к заказу» с ключом `comment`.

#### Скрытое число
Числовое поле, видимое **только в админке**. Не отображается на сайте, но участвует в расчётах.

| Параметр | Описание |
|----------|----------|
| Ключ поля | Уникальный идентификатор |
| Подпись | Название (не отображается на сайте) |
| Значение | Числовое значение (например, базовая цена) |

**Пример:** Базовая стоимость печати с ключом `base_price`, значение = 1500.

> **Зачем нужно:** Администратор может задать фиксированную цену или коэффициент, который не должен видеть пользователь.

### Переменные полей в формуле

| Тип поля | Переменная `{{key}}` | Переменная `{{key_value}}` |
|----------|---------------------|---------------------------|
| Число, Слайдер | Числовое значение | — |
| Выпадающий список, Радио | **Цена** выбранного варианта | Текстовое значение |
| Чекбокс | Цена если отмечен (0 если нет) | 1 или 0 |
| Текст | — | Введённый текст |
| Скрытое число | Значение из админки | — |

## 5. Формула расчёта

Формула определяет, как вычисляется итоговая стоимость на основе значений полей.

### Базовый синтаксис

Переменные указываются в двойных фигурных скобках:

```
{{pages}} * {{base_price}} + {{cover_type}} + {{gift_box}}
```

### Арифметические операторы

| Оператор | Описание | Пример |
|----------|----------|--------|
| `+` | Сложение | `{{price}} + 100` |
| `-` | Вычитание | `{{price}} - 50` |
| `*` | Умножение | `{{quantity}} * {{price}}` |
| `/` | Деление | `{{total}} / 2` |
| `()` | Скобки | `({{a}} + {{b}}) * {{c}}` |

### Математические функции

| Функция | Описание | Пример |
|---------|----------|--------|
| `min(a, b)` | Минимум | `min({{price}}, 1000)` |
| `max(a, b)` | Максимум | `max({{discount}}, 10)` |
| `pow(base, exp)` | Степень | `pow({{size}}, 2)` |
| `round(x)` | Округление | `round({{price}})` |
| `ceil(x)` | Округление вверх | `ceil({{area}})` |
| `floor(x)` | Округление вниз | `floor({{discount}})` |
| `abs(x)` | Модуль | `abs({{diff}})` |

### Условная функция `if()`

```
if(условие, значение_если_истина, значение_если_ложь)
```

**Операторы сравнения:**

| Оператор | Описание | Пример |
|----------|----------|--------|
| `>` | Больше | `{{qty}} > 10` |
| `<` | Меньше | `{{qty}} < 100` |
| `>=` | Больше или равно | `{{qty}} >= 10` |
| `<=` | Меньше или равно | `{{qty}} <= 100` |
| `==` | Равно | `{{type}} == 1` |
| `!=` | Не равно | `{{type}} != 0` |

**Логические операторы:**

| Оператор | Описание | Пример |
|----------|----------|--------|
| `&&` | И (оба истинны) | `{{qty}} >= 10 && {{price}} < 200` |
| `\|\|` | ИЛИ (хотя бы одно) | `{{vip}} == 1 \|\| {{qty}} >= 50` |
| `!` | НЕ (инверсия) | `!{{discount}}` |

**Вложенные `if()`:**

```
if({{qty}} >= 100, 80, if({{qty}} >= 10, 100, 150))
```

### Пользовательские переменные

Можно создавать промежуточные переменные для сложных расчётов. Каждая переменная указывается с новой строки и заканчивается `;`. Основная формула — последняя строка **без** `;`.

**Синтаксис:**
```
имя_переменной = выражение;
имя_переменной = выражение;
основная_формула
```

**Пример — оптовая цена:**
```
base_price = if({{kolvo}} >= 100, 120, 150);
{{kolvo}} * {{base_price}}
```

**Пример — скидка от суммы:**
```
sum = {{kolvo}} * {{price}};
discount = if({{sum}} > 10000, 0.15, if({{sum}} > 5000, 0.1, 0));
{{sum}} * (1 - {{discount}})
```

**Пример — сезонный коэффициент:**
```
season_coef = if({{month}} >= 5 && {{month}} <= 8, 1.2, 1);
{{base}} * {{season_coef}}
```

**Пример — скидка для VIP или оптовиков:**
```
discount = if({{is_vip}} == 1 || {{kolvo}} >= 50, 0.2, 0);
{{sum}} * (1 - {{discount}})
```

### Тестирование формулы

В блоке формулы есть кнопка **Тест**. Она позволяет проверить формулу с тестовыми значениями полей и увидеть результат расчёта.

## 6. Размещение на сайте

### Шорткод: Все продукты во вкладках

Отображает все продукты в виде вкладок:

```
[fk_calculator_tabs]
```

### Шорткод: Один продукт

Отображает конкретный продукт по slug:

```
[fk_calculator_product slug="butterfly-photo"]
```

### Как добавить на страницу

1. Откройте нужную страницу в редакторе
2. Добавьте блок **Шорткод**
3. Вставьте один из шорткодов выше
4. Сохраните страницу

## 7. REST API

Плагин предоставляет REST API endpoints для интеграции с внешними системами.

### Получить список продуктов

```
GET /wp-json/fk-calc/v1/products
```

### Получить конфигурацию продукта

```
GET /wp-json/fk-calc/v1/products/{slug}
```

### Расчёт стоимости

```
POST /wp-json/fk-calc/v1/calculate
Content-Type: application/json

{
  "product_slug": "butterfly-photo",
  "fields": {
    "pages": 20,
    "cover_type": "Твёрдая",
    "gift_box": "on",
    "quantity": 2
  }
}
```

**Ответ:**
```json
{
  "success": true,
  "price": 3500,
  "expression": "((20) * (1500) + (500) + (300))",
  "error": null
}
```

## 8. Примеры

### Пример 1: Фотокнига с выбором обложки

**Поля:**

| Ключ | Тип | Подпись | Настройки |
|------|-----|---------|-----------|
| `pages` | Число | Количество страниц | min=10, max=100, step=1 |
| `cover_type` | Выпадающий список | Тип обложки | Мягкая: 0, Твёрдая: 500, Премиум: 1500 |
| `gift_box` | Чекбокс | Подарочная упаковка | Цена: 300 |

**Формула:**
```
{{pages}} * 150 + {{cover_type}} + {{gift_box}}
```

### Пример 2: Оптовая печать с скидками

**Поля:**

| Ключ | Тип | Подпись | Настройки |
|------|-----|---------|-----------|
| `kolvo` | Число | Количество | min=1, max=1000, step=1 |
| `price` | Скрытое число | Цена за шт. | Значение: 200 |
| `is_vip` | Чекбокс | VIP клиент | Цена: 0 |

**Формула:**
```
sum = {{kolvo}} * {{price}};
discount = if({{is_vip}} == 1 || {{kolvo}} >= 50, 0.2, if({{kolvo}} >= 20, 0.1, 0));
{{sum}} * (1 - {{discount}})
```

### Пример 3: Фотокнига с форматом и разворотами

**Поля:**

| Ключ | Тип | Подпись | Настройки |
|------|-----|---------|-----------|
| `spreads` | Слайдер | Количество разворотов | min=5, max=50, step=1, default=10 |
| `format` | Радио-кнопки | Формат | 20×20: 0, 20×30: 300, 30×30: 600 |
| `base_price` | Скрытое число | Базовая цена | Значение: 1500 |

**Формула:**
```
{{base_price}} + {{spreads}} * 200 + {{format}}
```

## 9. Частые вопросы

### Как изменить порядок полей?
Перетащите поле за иконку ☰ в нужную позицию.

### Как удалить поле?
Нажмите на иконку 🗑️ в заголовке поля.

### Почему формула не работает?
1. Проверьте баланс скобок `(` и `)`
2. Убедитесь, что все `{{key}}` соответствуют ключам полей
3. Проверьте, что пользовательские переменные заканчиваются на `;`
4. Основная формула должна быть **последней** строкой без `;`
5. Используйте кнопку **Тест** для проверки

### Что такое скрытое число?
Это поле, которое не видно пользователю на сайте. Администратор задаёт в нём числовое значение (например, базовую цену), которое используется в формуле расчёта.

### Можно ли использовать формулу в нескольких продуктах?
Да, каждый продукт имеет собственную формулу. Скопируйте текст формулы из одного продукта в другой.

### Как работает скидка?
Используйте функцию `if()` в формуле или пользовательских переменных. Пример:
```
discount = if({{sum}} > 5000, 0.1, 0);
{{sum}} * (1 - {{discount}})
```

### Где посмотреть логи ошибок?
Ошибки формул записываются в PHP error log сервера. Также ошибки отображаются при тестировании формулы в админке.
